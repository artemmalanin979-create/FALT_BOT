🚀 ИНСТРУКЦИЯ ПО ДЕПЛОЮ
═══════════════════════════════════════════════════════════════════════════════

ШАГ 1: Миграция базы данных
   $ python migrate.py

   Добавляет:
   • Колонку email в таблицу users
   • Таблицу mini_app_sessions

ШАГ 2: Настройка переменных окружения
   Добавьте в .env:

   MINI_APP_URL=https://your-domain.up.railway.app
   JWT_SECRET_KEY=your_secret_key_here_32_chars_min

   Генерация ключа:
   $ python -c "import secrets; print(secrets.token_hex(32))"

ШАГ 3: Деплой на Railway (рекомендуется)
   1. Подключите GitHub репозиторий к Railway
   2. Railway автоматически найдет railway.json и Procfile
   3. Установите переменные окружения в Dashboard
   4. Получите домен (например, https://falt-bot.up.railway.app)
   5. Обновите MINI_APP_URL этим значением

ШАГ 4: Настройка BotFather
   /mybots → Ваш бот → Bot Settings → Menu Button → Web App

   Название: 🧺 FALT App
   URL: https://your-domain.up.railway.app

ШАГ 5: Тестирование
   1. /start → Главное меню (как раньше)
   2. /setemail → Ввод email → ✅ Подтверждение
   3. /miniapp → Кнопка "Открыть Mini App"
   4. Нажать кнопку → Mini App открывается
   5. В Mini App: выбрать машинку → дату → время → Забронировать

⚠️ ВАЖНЫЕ ЗАМЕЧАНИЯ
═══════════════════════════════════════════════════════════════════════════════

1. ОБЯЗАТЕЛЬНЫЕ ПОЛЯ:
   • MINI_APP_URL должен быть HTTPS (требование Telegram)
   • JWT_SECRET_KEY минимум 32 символа
   • Пользователь должен быть зарегистрирован в боте
   • Должен быть привязан email для использования Mini App

2. ОГРАНИЧЕНИЯ:
   • Mini App работает только в мобильном Telegram
   • Требуется интернет-соединение
   • Бронирование требует достаточного баланса

3. ОТКАТ ИЗМЕНЕНИЙ:
   Если нужно отключить Mini App:
   • Удалить импорты email_router и mini_app_router из bot.py
   • Удалить команды из set_commands()
   • Оставить database/db.py (не мешает работе бота)
